\textbf{The Uniform Distribution}
When $X$ is a uniformly distributed random variables, we would describe it mathematically as follows
\[ X \sim U(a,b) \]

\begin{description}
\item[a] The lowest possible value of $X$ (i.e. the minimum)
\item[b] The highest possible value of $X$ (i.e. the maximum)
\end{itemize}
